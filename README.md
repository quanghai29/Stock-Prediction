# Stock-Prediction
 

# Command Line Guide
 ## Lần đầu clone
    Cài đặt tất cả gói thư viện yêu cầu
       pip install "tên thư viện"
 ## Đối với jupyter
    sau khi đã cài đặt jupyter chạy câu lệnh
       python -m notebook
 ## Đối với dash
    terminal để đường dẫn ở thư mục root
       python stock_app.py
